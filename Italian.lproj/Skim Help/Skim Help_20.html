Changing the color for notes and highlights

Skim allows you can change the color of a note, highlight or line you have added using Skim. You can also change the default colors that are used for new notes, highlights and lines.

To change the default color for notes, highlights and lines:

  • Choose Skim > Preferences, select Notes, and click a color for the type of note, highlight or line you want to change.

To change the color of a single note, highlight or line:

  • Select the note, highlight or line in the PDF and choose Tools > Show Colors.
