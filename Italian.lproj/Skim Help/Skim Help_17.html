Adding notes to a PDF document

You can add your notes to a PDF document in boxes that look like paper sticky notes, or more extensive anchored notes that can be viewed in a separate window by double-clicking.

The notes are saved with the document, so also other Skim users can see them. The notes are not saved in the PDF data itself, therefore other PDF viewers will probably not be able to show the notes.

To add a text note:

  1. Choose Tools > Text Tool.
  2. Choose Notes > Text Note.
  3. Drag in the document until the note is located where you want and has the size you want.
  4. To enter text, double-click the note.

To add a longer text note:

  1. Choose Tools > Text Tool.
  2. Choose Notes > Anchored Note.
  3. Drag in the document until the note is located where you want.
  4. To enter text, double-click the note. A window will show containing the text of the note. It has a short text that is used for identification, and a longer text that can also contain attributes.
  5. You can also add a simple image to the note. Drag an image to the image well in the window for the note.

You can also use toolbar buttons to add notes. To add notes buttons to the toolbar, choose View > Customize Toolbar and drag the buttons to the toolbar.

You can edit or delete the notes. To move or resize the note, choose Tools > Text Tool, and use the mouse. Drag the black box to resize. To edit the note's text, choose Tools > Text Tool, and double-click the note. To change the color used for the note, choose Tools > Show Colors. To delete the note, choose Tools > Text Tool, select the note, and press Delete.
