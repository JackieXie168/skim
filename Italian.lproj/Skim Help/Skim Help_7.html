Finding text in a PDF file

You can search a PDF file for a word or any set of characters.

To find all occurrences of text:

  1. Open the PDF file. If the Contents pane isn't open, choose View > Show Contents Pane.
  2. Type some text in the search field and press Return. A list of places where the text occurs appears in the contents pane.
  3. Click an occurrence of the text in the Contents pane. Skim shows the page where the text occurs, and highlights the text.

To find a single occurrences of text:

  1. Choose Edit > Find > Find..., to show a Find panel.
  2. Type some text in the search field, and press Enter or hit Next.
