Changing the font of text notes

Skim allows you can change the font of a text note you have added using Skim. You can also change the default font that are used for new text notes.

To change the default font for text notes:

  • Choose Skim > Preferences, select Notes, and select a font name and size below 'Text note font'.

To change the font of a single text note:

  • Select the note in the PDF and choose View > Font > Show Fonts.
