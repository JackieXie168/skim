Mailing list

There is a mailing list set up for discussions of using BibDesk. Information is available at http://lists.sourceforge.net/mailman/listinfo/bibdesk-users.

Bug reports

You can submit a bug report to the bug tracker at http://sourceforge.net/projects/bibdesk/. If you're submitting a problem with how it handled a certain file, it would help if you sent that file too, or at least the representative subset that causes the errors. Crash reports are also useful and can be obtained from the Console application (Applications/Utilities).
