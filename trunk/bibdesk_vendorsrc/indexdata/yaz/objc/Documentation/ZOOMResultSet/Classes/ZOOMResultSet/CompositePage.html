ZOOMResultSet

Interface for ZOOM_resultset primitive object. Provides facilities for getting each record individually or in a batch. In general, this object will only be created by a ZOOMConnection or other object that owns a result set.

recordAtIndex:

Extracts each record individually. 

private

- (ZOOMRecord *)recordAtIndex:(unsigned int)index;
Parameters
index
Index of the record to extract. Must not exceed the number of available records.
Return Value

A ZOOMRecord instance. The caller is responsible for retaining this object.

Discussion

This may cause a network connection and read each time it is called, so it should only be used for a small number of records.

recordsInRange:

Returns all records in the specified range. 

private

- (NSArray *)recordsInRange:(NSRange)range;
Parameters
range
Range of records to return.
Return Value

An array of ZOOMRecord instances. The caller is responsible for retaining this object.

Discussion

This is the most efficient way to get a subset of the available records (rather than recordAtIndex:).
