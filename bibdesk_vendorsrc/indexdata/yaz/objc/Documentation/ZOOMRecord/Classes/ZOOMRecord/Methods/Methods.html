rawString

Converts the underlying data in its "raw" syntax to an NSString, using the character set supplied at initialization time. 

private

- (NSString *)rawString;
Discussion

The directory information for character counts in a MARC record may not be correct if you use this accessor. It converts the octets to a Unicode string, and the returned string length is not identical to the data length unless the record is entirely ASCII.
