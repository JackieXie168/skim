ZOOMConnection

This is the primary interface with the ZOOM objects.

Superclass: NSObject
Declared In: ZOOMConnection.h
Discussion

ZOOMConnection allows you to create a connection, set various options, and fetch results for a given query. All instance variables are private, and should not be relied upon. A new ZOOMConnection instance should be created for a given host, which allows cleaner caching and option handling. The connection is not established until results are asked for.
